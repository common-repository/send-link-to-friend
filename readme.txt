Send link to friend WordPress plugin will create a simple form in the website to send the URL. If user thought the content is useful to their friend, they can use this form to send the URL instead of copy and paste the URL into email. This form have the option to enter the email address and small description text box. the URL of the page automatically added into the email. also we have captcha option in the form to protect the spam mails. there are three ways to implement the form into the website.

= Feature of the plugin =		

1. Simple.  
2. Admin can choose which page this plugin should display.  
3. Easy style-override.
4. Widgets, so you can add pretty much anything.
5. Easy installation.
6. Ajax, Thus no page refresh.
7. reCaptcha option.
8. Option to add the form into pages/posts
